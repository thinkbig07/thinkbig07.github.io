Pixel 3 XL 都提前开卖了，下周的 Google 发布会还剩什么秘密？

科技 爱范儿 · Beats of Bits 2018/10/07

再过 3 天，Google 就将在纽约举办主题为「Made by Google Launch Event」的新品发布会，但随着 Pixel 3 等硬件新品信息的大量泄露，这场发布会似乎已经没有太多秘密了。
最近 Engadget 甚至直接在香港一家手机零售商处购得一部包装完整的 Pixel 3 XL，据称花费了15880 港元（约合 2030 美元）。

其实早在 8 月，乌克兰地区就有用户放出了 Pixel 3 XL 的真机实拍图，还有一家俄罗斯网站做了简单上手， Google 为此还专门派人前往富士康代工厂调查泄露的源头。而这次真机更直接泄露到了渠道商上，这种程度的泄露可以说前所未见了。

▲ 图片来自：ROZTKED
Engadget 拿到的真机也确认了从前不少爆料的真实性，外观和曝光的一些真机实拍几乎一致。虽然背部依旧是标志性的「撞色拼接」，但转身后的「宽刘海」和「大下巴」多少还是有点让人失望。


不过这次泄露的是黑色版本，电源键并没有采取撞色设计。但根据此前一些爆料，Pixel 3/Pixel 3 XL 会有黑、白、薄荷绿、淡粉四种配色，其中浅色的 Pixel 的电源键将继续采用撞色设计。


▲Pixel 3 XL 假想图. 图片来自：Phonearena
再来看看配置， 这部 Pixel 3 XL 采用了骁龙 845 处理器，4 GB 运存，128 GB 内存，运行 Android 9 系统，不过 Engadget 编辑认为会有更高运存的版本，但这个配置在 2018 年的 Android 旗舰中实在没有太多出彩的地方。

此外 Pixel 3 XL 配备了一块 6.3 英寸、分辨率为 2960 x 1440 的 OLED 屏幕，而 Engadget 用 CPU-Z 测试得出的电池容量为 3732 mAh，并支持无线充电。

拍照方面，跟之前的传言一样，Pixel 3 XL 依旧没有采用后置双摄，而搭载是一块 1220 万像素的单摄像头，前置摄像头则采用了 800 万像素的双摄。

尽管后置单摄在 2018 Android 旗舰中有点「落伍」了，但此前外媒将 Pixel 3 XL 与采用后置双摄的 Galaxy Note 9、华为 P20 Pro 的样张对比，凭借 Google 的 AI 虚化算法，Pixel 3 XL 的拍照水准依旧不落下风。



▲ 从上往下依次是 Pixel 3 XL、华为 P20 Pro、Galaxy Note 9
但这样的 Pixel 3 XL 在目前竞争激烈的智能手机市场中的位置略显尴尬，就像爱范儿在之前一篇文章里说的：

仅仅是主打原生 Android 和人工智能，套上其它品牌都在用的元器件配置，Pixel 手机还很难找到让大众消费者掏钱买单的理由。
至此， Pixel 3 系列已经几乎没有任何悬念，所以在下周的发布会 Google 还能拿出什么新东西呢？可能是 Pixelbook 2 、Pixel Slate 平板、 Google Home Hub 智能音箱等硬件吧，不过这些产品的曝光同样不少。

外媒 MySmartPrice 今天放出了 Pixel Slate 的泄露图像，Pixel Slate 会是 Google 首款运行 Chrome OS 的平板电脑，将搭载 3：2 的触摸屏，配备立体声扬声器和 USB Type-C 端口、前置/后置摄像头，顶部电源键内置指纹识别模块。

Pixel Slate 也配备了键盘底座，二合一后可作为笔记本电脑使用，但与 Surface Pro 不一样的是键盘采用了打字机式样的圆形键帽。目前看来，Pixel Slate 或许将和 Surface Pro 和 iPad Pro 系列直接竞争。

而 Pixel Slate 也和 Pixelbook 一样支持手写笔，可能会搭配新一代的 Pixelbook Pen，最近外媒也泄露了白色和深蓝色的新 Pixelbook Pen，这可能也是 Pixel Slate 的配色。

▲图片来自：9to5google
而新一代 Pixelbook 应该也会在下周的发布会上亮相，爆料大神 Evan Blass 曾透露 Pixelbook 2 将会有更窄的边框，会是有一个尺寸更小的版本，同时支持更多 Android 应用。

除了手机、平板和笔电，Google 还可能发布一款新的智能音箱。上个月 MySmartPrice 曝光了一款名为 Google Home Hub 的设备，这台智能音箱配备一块 7 英寸 LCD 触摸屏，可以让用户查看天气和通勤等信息，还能创建提醒日程。

▲ Google Home Hub 渲染图
虽然目前 Google 的硬件组合已经十分丰富，但是对比苹果成熟的软硬件生态，Google 还是稍逊一筹。而在新硬件「全面曝光」后，下周发布会的惊喜或许只能寄希望于软件层面上的更新了。

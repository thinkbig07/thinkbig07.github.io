Google 发布会最全汇总：Pixel 3 暗光拍摄吊打 iPhone Xs 

科技 爱范儿 · Beats of Bits 2018/10/10

原以为今年新 iPhone 发布前真机图流出已经是十分严重的泄露，但 Google Pixel 3 XL 手机整套机器提前被买到手可以说是刷新了新机泄露的记录了。

▲提前被买到的 Pixel 3 XL
今天 Google 新品发布会在纽约如期召开，或许因为 Pixel 3 系列已经没有太大悬念， Pixel 3 被安排到最后才登场，但这款新机依然是这场发布会的主角。
尽管 2017 年 Pixel 的销量还不到 400 万部，但 Pixel 不仅寄托了 Google 进军高端智能机市场的期望，也是其软硬件生态中一块重要的拼图。

▲这次发布会的几款新品
去年的发布会上，Google 提出了「 AI + 软件 + 硬件」的战略，5 月份的 Google I/O 大会上 AI 的戏份最重，AI 算法也贯穿在这次发布的几款新品中，为已经没有秘密的 Pixel 3 系列撑起了为数不多的亮点。

没有秘密的 Pixel 3 /3 XL，拍照依旧是最大亮点

就像之前曝光的真机一样，Google Pixel 3 /3 XL 依然沿用了前代标志性的撞色设计，但 3 XL 首次采用了「刘海」全面屏，边框更薄，边角更圆，同时也保留了较宽的的「下巴」，这样的外观在 2018 年的旗舰机中着实不太讨喜。

▲ Pixel 3 XL
Pixel 3 则没有采用刘海设计，看起来就像是缩小版的 Pixel 2 XL。

▲Pixel 3
不过两款新机背部不再采用金属+玻璃的材质，而是完全由玻璃组成，但下半部分采用的是磨砂玻璃，这也意味着 Pixel 3 会比 Pixel 2 更重一些。
Google Pixel 3 /3 XL 共有黑、白、粉三种配色，其中粉色和白色的电源键仍然采用了撞色设计。

两款新机的配置也基本相同：
处理器：高通骁龙 845

  • 运行内存：4GB RAM
  • 存储空间：64GB / 128GB
  • 后置摄像头：12.2 MP
  • 前置摄像头：8 MP 双摄
  • 前置双扬声器
  • 支持「Active Edge」唤醒 Google Assistant
  • USB-C 接口，支持无线充电
  • 支持蓝牙 5.0
  • 18W 快充
  • IP68 防尘防水
    Pixel 3 的屏幕尺寸为 5.5 英寸，分辨率 2160 x 1080，而 3 XL 则搭载一块 6.3 英寸、分辨率为 2960 x 1440 的 QHD + 屏幕。

    续航方面，Pixel 3 的电池容量为 2915 mAh，3 XL 为 3430 mAh。除了能通过 USB-C 进行 18W 快充外，同时支持无线充电,Google 也为此推出了一个无线充电支架 Google Pixel Stand 。

    Pixel 3 预装了Android 9.0 系统 ，UI 中提供了手势导航操作，也继承了之前的 Active Edge 侧边挤压捷径功能，可以通过挤压手机两侧就能让来电静音，或者唤醒 Google Assistant。
    拍照功能依然是 Pixel 3 系列最大的亮点，尽管依旧没有采用后置双摄，但上了前置双摄，采用标准镜头+超广角镜头的组合。而这两颗摄像头藏在刘海中，所以 Pixel 3 XL 的刘海才会这么宽。

    去年的 Pixel 2 利用 AI 算法，凭借单摄像头拍出不输双摄的人像效果，一度以 98 分位居 DXO Mark 拍照评分第一位。

    ▲现在第一的位置是华为 P20 Pro，为 109 分
    跟 Pixel 2 一样，Pixel 3 也主要利用 AI 算法来提升拍照质量。新推出的 Top Shot 功能可以在一系列连拍中选取最佳的一张，将眨眼虚焦这样的废片剔除。而 Super Res Zoom 功能则可以在变焦时填充画面细节。

    此外，Pixel 3 还新增了Photobooth 自动抓拍、Motion Auto Focus 自动追踪对焦和 Night Sight 夜景增强、Playground AR 贴纸等功能，并跟前代一样提供三年无限量储存的 Google Photos 服务。

    其中 Pixel 3 Night Sight 夜景增强功能令人印象深刻，发布会上 Google 展示了 Pixel 3 与 iPhone Xs 的暗光条件下拍摄样张对比，从展示效果来看 Pixel 3 可以说是吊打 iPhone Xs 。

    由于 Pixel 3 早已经被泄露个精光，一些媒体已经都提前做了Pixel 3 的上手评测，甚至跟其他旗舰机进行了拍照横评。
    ROZTKED 将 Pixel 3 XL 与采用后置双摄的 Galaxy Note 9、华为 P20 Pro 的样张对比，Pixel 3 XL 的拍照水准依旧不落下风，在动态范围、解析力上的表现不错。



    ▲ 从上往下依次是 Pixel 3 XL、华为 P20 Pro、Galaxy Note 9
    UNWIRE.HK 则将 Pixel 3 XL 与 iPhone XS Max 的拍照进行对比，结果显示在室内弱光环境下，iPhone XS Max 比较能还原灯光的暖色调，而 Pixel 3 XL 的色调偏冷。


    ▲上为 Pixel 3 XL ，下为 iPhone XS Max
    不过在室外的夜景中， Pixel 3 XL 对暗部细节的控制较好，而 iPhone XS Max 则稍显过曝，由此可见夜景拍摄确实是 Pixel 3 系列的强项。


    ▲上为 Pixel 3 XL ，下为 iPhone XS Max
    至于两款新机的价格，Google Pixel 3 64GB 售价 799 美元，128GB 售价 899 美元。Google Pixel 3 XL 64GB 售价为 899 美元，128GB 售价为 999 美元。Google Pixel Stand 无线充电底座售价 79 美元。

    Pixel 3 从今天开启预售，并于 10 月 18 日发货，将在美国、加拿大、日本等 13 个国家和地区推出，其中不包括中国内地，购买的用户将免费获得 6 个月的 YouTube Music 付费会员服务。

    Pixel Slate ：首款运行 Chrome OS 的平板电脑

    虽然传言中的 Pixelbook 二代没有出现，但 Google 首款运行 Chrome OS 的平板电脑 Pixel Slate 来了。

    先看看配置：
  • 处理器：第 8 代英特尔酷睿 m3 / i5 / i7 / Celeron 处理器
  • 内存及存储空间：4GB/8GB/16GB/ + 32GB/ 64GB/128GB/256GB/。支持 NVME 固态硬盘
  • 电池：48 瓦时，续航时间达 10 小时
  • 前置摄像头：800万像素，f/1.9 光圈，单位像素尺寸 1.4 μm，可录制 30fps 的 1080p 视频
  • 后置摄像头：800万像素，f/1.8 光圈，单位像素尺寸 1.12 μm，自动对焦，可录制 30fps 的 1080p 视频
  • 支持 WIFI 及蓝牙 4.2
  • 支持 Google Assistant，Google Assistant 专门按键
  • 两个 USB-C 接口,取消 3.5mm 耳机插孔
  • 机身厚度为 7 mm，重 0.7 kg
  • 顶部电源键内置指纹识别模块
  • 支持 Pixelbook Pen
    Google 的高级产品经理 Ben Janofsky 在发布会上介绍， Pixel Slate 将同时满足工作和娱乐两个使用场景。并表示这是「市场上首款真正没有妥协的设备」。

    Pixel Slate 配备了一块 12.3 英寸、分辨率为 3000 x 2000 的屏幕，Google 表示 293 ppi 的像素密度是同类型产品中最高的。

    Pixel Slate 颜色为午夜蓝，机身采用了阳极氧化处理的铝材质，因此厚度仅有 7 mm。重量约 0.7 kg ，十分轻便。
    Pixel Slate 也配备了新的键盘底座，二合一后可作为笔记本电脑使用，但与 Surface Pro 不一样的是键盘采用了打字机式样的圆形键帽，还内置键盘背光灯，但键程就只有 1.2mm 。

    而且新的键盘底座采用了磁吸式脚架，可以实现多角度的摆放。

    同时 Google 也为 Pixel Slate 设计了新的 Chrome OS 系统，根据是否连接鼠标和键盘运行两套不同的 Chrome OS 系统。连接上时运行传统风格的 Chrome OS ，断开时则切换为 Android 风格的界面，可分屏显示还加入了手势操作，Pixel Slate 也同样支持 Android 应用。

    目前看来，Pixel Slate 或许将和 Surface Pro 和 iPad Pro 系列直接竞争，在价格上也比后两者有一定优势。

    此外 Google 还为 Pixel Slate 更新了一款相同配色的 Pixelbook Pen ，延迟更短，续航时间更长，由一个AAAA电池（9号碱性电池）供电，但需要一个双面胶黏扣才能固定在 Pixel Slate 上。

    Pixel Slate 起售价为 599 美元，顶配版价格为 1599 美元。另外配套的键盘售价 199 美元，Pixelbook Pen 99 美元，Pixel Slate 用户将获得 3 个月的 YouTube TV 试用期。

    Google Home Hub ：带屏幕的智能音箱

    Google 还带来了一款智能显示器 Google Home Hub ，其实就是一台带屏幕的智能音箱。这类产品不算新鲜，亚马逊的 Echo show 就是其中的代表，Facebook 也在昨天推出了类似的产品。

    Home Hub 配备一块 7 英寸 LCD 触摸屏，有白、黑、薄荷绿、粉红四种配色，可以让用户通过语音查看天气和通勤等信息，还能创建提醒日程。

    这款产品出自 Google 的智能家居品牌 Nest，可以成为智能家居的中枢。可以通过 Ambient EQ 功能自动调节室内的亮度和色温，以及通过 Home View 远距离查看家中情况。

    Home Hub 也可以作为相框轮播照片，跟 Pixel 3 的 Top Shot 功能类似，Home Hub 可以自动排除一些虚焦或曝光不足的照片，还能根据用户需求展示指定人物的照片。

    跟 Facebook 一样，Google 最近也身陷用户泄露丑闻中， Home Hub 上也体现了 Google 对于隐私保护的态度。Home Hub 没有内置摄像头，还有提供一个可关闭麦克风的物理按键，以保证用户隐私不被泄露，但也意味着这款设备不支持视频聊天。
    此外 Home Hub 也支持 YouTube 和 Google Assistant 等 Google 系应用，但数量不多，Netflix、Hulu、Amazon Prime 等热门视频应用均不支持。

    Google Home Hub 的价格为 149 美元，购买者可获赠 6 个月 YouTube Premium 会员服务。

    总的来说，Google 这次的发布会没有太多眼前一亮的东西。不只是因为前所未有的泄露事故，产品本身也比较中规中矩。
    虽然 Pixel 3 的拍照依然不乏亮点，但是 4GB 运存和后置单摄的配置在 2018 年 Android 旗舰中其实也不太拿得出手了。

    这样的 Pixel 3 在目前竞争激烈的智能手机市场中的位置略显尴尬，就像爱范儿在之前一篇文章里说的：

    仅仅是主打原生 Android 和人工智能，套上其它品牌都在用的元器件配置，Pixel 手机还很难找到让大众消费者掏钱买单的理由。
    而 Pixel Slate 可能是一款不错的入门款二合一平板，低配版的价格比起 iPad Pro 和 Surface Pro 都更有吸引力，但在同等配置下就没有价格优势了，而 Home Hub 和同类产品相比几乎没有什么不同。

    尽管《连线》杂志把 Pixel 称为 「2018 年可以买到的最好的智能手机」，但不可否认对比苹果、微软等厂商，硬件仍是 Google 的短板。

    Google 在发布会上多次强调的「 AI + 软件 + 硬件」战略或许没错，但前提是要把硬件的短板补上，而这条路还有一段不短的距离。

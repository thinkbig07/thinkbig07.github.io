湖南移动NB-IoT正式商用

四川首个大规模NB-IoT智能水表项目落地成都

7月26日，来自中国电信成都公司的消息表明，天府新区安置房智能水表改造项目一期工程顺利完工，通过两个月试运行，首批安装上线的8236支NB-IoT智能水表抄表成功率稳定在99.6%以上，抄表准确率达100%。
这是四川省首个NB-IoT智能水表规模应用圆满落地，标志着天府之国正式迈入智能水表时代。随着车联网、智慧消防、智慧农业等应用不断发展，截至目前，成都电信物联网规模已突破100万。

IDC预测智慧城市支出将在2022年达到1580亿美元

根据国际数据公司（IDC）新发布在《全球半年度智慧城市支出指南》的数据，智慧城市计划（Smart City initiatives）2018年里将在全球吸引超过810亿美元的技术投资，而 2022年的支出将增长到1580亿美元。该《支出指南》对一系列与智慧城市优先级、计划和用例相关的技术投资做了详细介绍。
三个最大的用例是固定性视觉监控、高级公共交通和智慧户外照明，这三大用例将在2018年吸引全球智慧城市近四分之一的支出。到2022年时，智慧交通管理将超过智慧户外照明占第三位，届时前三大用例的支出仅占总支出的五分之一，小型和快速增长的用例将出现并成为核心要素。特别是办公人员可穿戴设备和车辆到一切（V2X）连接将出现最快的增长，尽管目前这两项在大多数地区都是从小的基础起步。

从地域上看，包括中国和日本在内的亚太地区将占全球2018年支出的近42％，其次是美洲（33％）、欧洲、中东和非洲（EMEA）（25％） ）。美国是智慧城市支出的最大国家市场（2018年超过230亿美元），其次是中国。三个最大用例在所有地区的支出都排在前五，而当前在大的智慧城市投资领域里出现的其他用例包括美国和拉丁美洲的移动视频捕获和记录以及中东和非洲地区的数字许可、牌照和检查。
目前IDC数据库里列出的53个城市支出约占全球智慧城市支出的15％，其中新加坡、东京、纽约、伦敦和上海在2018年的 投资处于领先地位。
（来源：IDC官网）

旷视科技“AI+IoT”进驻芜湖助推智慧城市建设

7月24日，旷视科技与芜湖市人民政府签署合作协议，旷视科技将在芜湖注册成立子公司，推动芜湖市人工智能产业发展。这也是安徽省人民政府发布《安徽省新一代人工智能产业发展规划（2018—2030年）》（以下简称《规划》）后，首家入驻的人工智能企业。

根据协议，旷视科技在落户芜湖后，将依托自身顶尖的技术研发水平及行业领先的落地方案，重点围绕智慧物流、智慧新零售、智慧城市、人工智能人才培养、人工智能产业基金等方面开展项目建设。
目前，旷视与芜湖市政府已在人工智能相关项目建设中达成多个共识，双方计划在芜湖成立智慧物流研究院，以推动芜湖仓储物流行业升级及智能装备制造产业发展；开展智能化便利店建设，树立全国新零售发展浪潮中的示范城市；建设城市超脑运算中心，打造国内标杆式“AI+IoT”智慧城市；共建人工智能学院，加速培养人工智能、大数据等专业技术型人才和行业应用型人才；成立芜湖旷视人工智能产业投资基金，吸引全国人工智能企业进驻芜湖、形成产业集聚效应。
（来源：中华网）

白城市加快建设“智慧城市”

白城市顺应城市发展规律，转变城市发展方式，在创新实施“海绵城市+老城改造”的同时，抢抓“数字吉林”建设机遇，融合打造“智慧城市”，构建了“一总七分”的城市智慧管理体系。
“一总”就是一个总的智能化综合管理指挥中心；“七分”就是公安、交警、城管、消防、社区、安监、市政“七大分中心”，全部建成运行后，可实现城区重点部位、重点区域视频监控统一管控无死角，信息互联互通、共享共用，有效解决城市管理中行业部门条块分割、数据孤岛问题。
目前，该市智慧交通已开始运行，共安装高清摄像头100余处、智能信号灯104处、LED信息诱导牌24块，区域内视频、流量监测覆盖率达90%以上，市区交通运行情况实现可视、可测、可控；城市管理依托“天网工程”的5000余个共享视频信号资源，利用150辆管理执法车和环卫作业车的GPS定位系统，以及执法记录仪、单兵手持终端，形成了“网格+数据”“指挥+监察”“执法+服务”于一体的城市管理新模式，推动城市管理方式实现了由粗放向精细、由低效向高效、由查处打击向执法服务“三个根本性转变”。
（来源：吉林省政府网）
盘锦市携手腾讯打造移动支付智慧城市
以雄安新区设立为标志，在全国范围内，智慧城市建设正在提速。7月9日，盘锦市政府与腾讯签署《智慧城市战略合作协议》，双方将携手落实国家《互联网+行动计划》，在商业、政务、民生、交通等领域建立全方位的“移动支付智慧城市”合作。
盘锦市中心医院、盘锦公交、盘锦市住房和城乡建设委员会、辽宁易安居信息技术有限公司等与腾讯分别签署智慧医疗、智慧交通、智慧社区合作协议，现场还对盘锦市中心医院进行了“人工智能医学示范基地“的授牌。盘锦市人民政府副市长白光、腾讯互联网+副总裁李致峰等出席签约仪式。
不只是更便利的支付，在商业、教育、政务、民生、医疗、交通出行等领域，盘锦市政府与腾讯将结合场景落地，探索基于支付升级的智慧城市解决方案。依托腾讯移动支付、大数据、云计算等技术优势，盘锦市将与腾讯携手打造新型智慧城市建设标杆。
（来源：腾讯网）
金山区制定智慧城市建设三年行动计划
为推动信息技术与城市发展全面深度融合，加快建设高水平信息基础设施，形成以数字化、网络化、智能化为主要特征的智慧城市，充分发挥信息化在加快本区打造“三区”“五地”和全面建设“三个金山”的重要作用，依据《上海市推进智慧城市建设 “十三五”规划》和《金山区国民经济和社会发展第十三个五年规划纲要》，金山区编制《金山区智慧城市建设三年行动计划（2018-2020年）》（以下简称《行动计划》）。
《行动计划》提出，通过三年持续推进，到2020年，金山区智慧城市建设基本形成以下一代信息基础设施、网络安全、数据资源开发共享为内涵的智慧城市支撑体系，以立体高效的城市管理、普惠宜居的智慧生活、协同透明的智慧政务为重点的智慧应用服务体系，以高端、绿色、融合为特征的智慧经济创新体系。初步建成以泛在化、融合化、智敏化为主要特征的智慧城市，智慧城市发展水平指数达到95，总体水平位居上海市郊区前列。
（来源：上海市政府网）
北川将建西南地区智慧城市新标杆
7月26日，在“智慧大美羌城——泛美航空科技城”项目的签约仪式上，中国电信绵阳分公司副总经理周春城在谈到北川未来的智慧生活时如数家珍。据其介绍，中国电信绵阳分公司将先期投入1000万元用于北川“智慧大美羌城”的建设，总投资将超过5000万元，全面助力大美羌城的全面数字化、信息化、智能化建设。项目建成后，“智慧大美羌城”将成为西南地区智慧城市的新标杆。

据了解，“大美羌城”——泛美航空科技城是集产、学、研、创、游、住、育、商为一体的科技新城，项目总投资50亿元，为北川史上最大招商引资项目。项目位于北川通用航空产业园内，毗邻北川新县城，由四川泛美教育投资集团投资建设，将按照“教育与文化旅游先行，科研与配套同步，产业孵化与制造跟进”的原则，分三期建成——一期将完成西南航空飞行学院主体建设，包括第一教学大楼、图书馆等，预计2018年完工并正式招生;二期主要完成学院第二教学大楼等建设，并启动创客公园和通航小镇建设;三期主要完成创客公园、飞行俱乐部、飞行员模拟训练中心等。值得一提的是，西南航空飞行学院校园建筑充分吸收各类建筑的艺术精髓，学院建成后将是5A级景区——北川羌城旅游区的重要组成部分。
（来源：四川在线）

阿里云IoT发布视频服务Link Vision

7月26日，阿里云IoT发布了云边端物联网视频服务Link Vision 1.0，该服务包含了物联网视频开放平台和物联网视频AI解决方案，可帮助传统摄像头厂商与安防工程全面实现数字化和智能化转型。据介绍，Link Vision 1.0将于8月底正式上线。

在以城市或大型企业为单位的安防监控系统中，传统解决方案无法实现海量视频有效存储和调阅，这导致企业无法实现复杂的视频AI结构化分析，也无法实现视觉对象和安全事件的全局搜索、布控、跟踪，一旦遇到安全事件，无法及时警告和有效后续跟踪。
阿里云Link Vision物联网视频AI解决方案可解决该问题。据介绍，该方案的边缘媒体服务器单机可以实现最多128路本地工业级网络摄像头的联网接入，企业能够提取高质量的视频AI结构化数据并上传至云端存储，轻松实现全局范围内的视觉对象、视觉事件的搜索、布控、跟踪等上层业务。
（来源：驱动之家）

Google推出AI芯片Edge TPU

7月25日，Google重磅推出了Edge TPU芯片，这款芯片将为边缘设备提供强大的计算和学习能力，或将成为Google边缘计算战略的重要组成部分。
据了解，Edge TPU芯片是功能强大的云TPU（Tensor Processing Unit）处理器的简化版本，是专门设计的加速器芯片，用于在边缘运行TensorFlow Lite机器学习模型，它将使传感器和其他设备能够更快地处理数据，并且极大突破了边缘计算设备的狭小空间和功率限制。

另外，Edge TPU可以自己运行计算，而不需要与多台强大计算机相连，因此应用程序可以更快、更可靠地工作，它们可以在传感器或网关设备中与标准芯片或微控制器共同处理AI工作。
Edge TPU将有助于吸引那些已经使用TensorFlow作为其主要机器学习框架的开发人员，他们的想法是能够更快更高效地运行这些模型。如果成功，将在TPU和TensorFlow级别上将这些开发人员进一步锁定到Google的云生态系统中。
（来源：雷锋网(公众号：雷锋网)）

雷锋网按：2018 全球人工智能与机器人峰会（CCF-GAIR）在深圳召开，峰会由中国计算机学会（CCF）主办，雷锋网、香港中文大学（深圳）承办，得到了深圳市宝安区政府的大力指导，是国内人工智能和机器人学术界、工业界及投资界三大领域的顶级交流盛会，旨在打造国内人工智能领域最具实力的跨界交流合作平台。
在大会第一天的「AI 前沿」专场，美国犹他州立大学计算机系终身教授、电子工程系兼职教授、计算机视觉与模式识别实验室主任承恒达带来了题为「Studying Uncertainty in Artificial Intelligence」的主题演讲。

承恒达教授于 1985 年获得普渡大学电子工程专业博士学位，师从国际著名专家、智能控制学科奠基人、模式识别学科创始人傅京孙（K.S. Fu）教授，他的主要研究方向包括模糊逻辑、计算机视觉、模式识别、图像处理，近年来专注于模式识别基础理论和乳腺癌早期诊断技术的研究。
承教授在演讲中讲解了人工智能中的不确定性，用通俗易懂的例子阐述了这一研究对人工智能领域一系列关键问题的推动意义。
他表示，人们在经典逻辑的基础上发展出微积分、代数、经济学、生物学等一系列学科，但是经典逻辑存在许多悖论，所以需要提出新的逻辑。「经典逻辑的值是 0 和 1，但模糊逻辑不一样，它的值在 0 和 1 之间，能解决一些经典逻辑不能解决的问题。模糊逻辑在科学和工业上也已经得到了许多的应用。」

很多人问我为什么要研究不确定性，原因很简单。不确定性遍存于宇宙中，不同的领域有不同的不确定性，研究不确定性能够很好地解决有关问题，从而得到好的结果。
如何处理不确定性？现在有多种不同的方法，比如说对于随机性可以用概率（probability）来解决，此外还产生了其他的不确定性，因此产生了其它的各种逻辑，如模糊逻辑（fuzzy logic）、粗糙集（rough set）、可能性理论（possibility）等。
建立在经典逻辑基础之上，人们发展出微积分、代数、经济学、生物学等等，可是哥德尔证明了科学是不完备的，经典逻辑存在许多悖论，所以需要提出新的逻辑。
经典逻辑的值是 0 或 1，不是黑就是白。模糊逻辑就不一样，它的值在 0 和 1 之间。比如说你在冰雪道上开车，不可能将刹车踩死或者干脆不踩刹车，所以绝不是按 0，1 这种逻辑来做。当然，模糊逻辑的提出已经得到了许多的应用，比如科学上、工业上，像手机聚焦就是利用模糊逻辑。

模糊逻辑隶属度在 0 和 1 之间，概率也是在 0 和 1 之间。很多人说，那这样的话，概率和模糊逻辑有什么差别？前者是建立在随机性的基础上，后者是建立在精确性的基础上。
举个例子来说明模糊逻辑和概率的不同。假定你在沙漠里必须得喝水才能存活下去，但水被污染了。假定致命水平是 0.4，你需要从两个盒子里选出一瓶水来喝，那你应该从下图中的哪个盒子里选择呢？每盒各有十瓶水，假定第一盒有两瓶被污染了，概率是 0.2，第二盒十瓶水均被污染了，隶属度是 0.2。这时候你应该从第二盒中拿水喝。

如果第一盒有四瓶水被污染，概率是 0.4，而第二盒水被污染的隶属度是 0.4，在这个基础上你从哪个盒子里选呢？从第一个盒子里选还有机会活下来。

所以模糊逻辑和概率是不一样的。如果用模糊逻辑作为一个子集，又可以发展出新的中智逻辑（Neutrosophic logic）。这个新的逻辑包含 I、T、F 三个集合，它们可在 0 和 1 之间取值。


如果从逻辑上看，经典逻辑是 0，1 两个孤立的点，模糊逻辑是一条线，中智逻辑可以考虑成一个球体，能解决一些模糊逻辑解决不了的问题。比如说两名审稿人评审同一篇学术论文，论文接收的隶属度都是 0.8，可是前者是专家，后者是新人，这两个 0.8 对文章的接收是有不同影响的。

可是在模糊逻辑下这两个是没有区别的。如果我们有不确定变量 I（indeterminate），就可以区分这两者。

比如由三维到二维变化的过程中存在信息丢失；另外有些定义本身就是模糊的，比如边缘（edge），对比度（image contrast）等；有时候没有一个客观标准来衡量问题，常常采用一些主观方法，比如投票等；用一些词汇来描述差异，比如说热，很热，特别热，一点点热等，这些可以用模糊逻辑解决得很好。
同时在医学影像中，医生可能会进一步引入主观性和模糊性。

在中国乳腺癌的发病率和死亡率都在增加，所以早期诊断就成为一个很重要的因素。我们开始研究是用 X 光，但因为 X 光具有放射性，后来我们就转到用超声，超声本身没有放射性，很方便，而且价格低廉，适合普查。除了 X 光，CT 也有一定的放射性，而且 CT 和 MRI 更贵一些，也不适合普查。

因为现在超声科医生很少，可以利用这一系统训练医学院的学生，将学生检测的结果和系统检测的结果进行比较就可以促进学生学习。而且系统能够提供很好的服务，这对于低资源地区和国家极有意义。如果有了这个系统，相当于超声机自带一个经验丰富的医生。在这里我们也用了一些方法来解决不确定性。最近我们正在把系统放在医院和体检中心，和他们做一些合作。

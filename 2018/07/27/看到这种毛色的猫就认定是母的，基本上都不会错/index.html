看到这种毛色的猫就认定是母的，基本上都不会错

今日热点 知乎日报 2018/07/27

这些都是母猫。



猫有两种色素，每种色素有两种浓淡，加上白色的底色就有很多种搭配了。
橘色是显性，黑色是隐形。

理论上来说猫应该是橘色，或者黑色。 然而实际上会出现玳瑁，三花这种不同区域的毛颜色不一样的现象。
这是因为编码色素的基因坐落在 X 染色体上，母猫有两条 X 染色体，当母猫携带一条橘色基因，一条黑色基因时，有的区域的皮肤细胞关闭了黑色基因所在的染色体，有的区域的皮肤关闭了橘色基因的颜色体，就变成了有的地方橘，有的地方黑。这个现象叫做 X 染色体失活(XIST)
1949 年，美国学者巴尔发现像细胞核染色后母猫的细胞有一个深色区而公猫没有， 这个深色区被命名为巴氏小体。

X- 染色体失活的主要意义在于，雄性只有一条染色体，雌性有两条，那么就会出现一个问题：如果没有 X 染色体失活，雌性因为携带更多的基因原本，就会产生更多的转录产物，对生命可能会有不利的影响。如果失活一条 X 染色体的话，就可以解决这个问题 。当需要发育第二性征时，可以两条 X 染色体都打开加速发育。。
有意思的是，X 染色体失活像是随机的。 所以猫的皮肤上的 X 染色体随机失活，就会导致毛囊随机表达橘色基因或者黑色基因，毛色呈现橘色或者黑色。
在胚胎发育的早期不同的细胞就确定下来哪条染色体失活了，然后细胞复制分裂长成一大片，毛色就是一块黑，一块黄而不会像手机 LED 屏幕一样，一根黑色一根橘色间隔排列。


更有意思的是，科学家克隆第一只猫时用的就是三花猫，然而克隆出来的产物是一只黑白双色猫……这就很尴尬了……
(左边是体细胞供体，叫做彩虹(Rainbow), 右边是克隆出的小猫，叫做 CC （copycat）)

截止至 2017 年，CC 和克隆她的科学家还都健在。当时 CC 已经 15 岁了，克隆她的科学家 Duane Kraemer 看起来 80 多岁。
比较奇怪的是，CC 的三个孩子都是黑色，没有橘色。由于样本量太小，不知道是巧合还是她的橘色基因所在的 X 染色体永久的失活了，在形成卵子的时候都没有恢复。。我没有再跟进后续研究了。

所以，得到的结论，
如果你喜欢的三花猫去了彩虹桥，不要想着能克隆出一只长得一毛一样的。
(备注，也有公的三花猫，不过概率非常非常低，因为他的染色体会是 XXY。)

34 个大类，超过 400 个「macOS 绝赞开源应用」列表

酷软推荐 小众软件 2018/07/03

这个在 GitHub 叫做 🚀 Awesome list of open source applications for macOS 的项目，收集了超多的 macOS 开源应用，大概有 34 个大类，超过 400 个应用。@Appinn

还记得 Windows 绝赞应用么？是一个精选 Windows 应用的推荐列表，涉及良心国产、音频文件、即时聊天、压缩处理、美化工具等 20 多个分类的，皆在给用户提供一个更好的软件选择、参考平台。
而这个 macOS 绝赞开源应用，特别强调收录的都是开源应用。虽然是 macOS 软件，但还是有很多的跨平台软件，所以都可以参考下。
不过鉴于内容太多，青小蛙也没有办法全部介绍，就大概摘录一下分类吧，有需要的同学可以前往 GitHub 研究。

      注意，这不是一个针对 macOS 初学者的软件推荐列表，这是一个仅仅收录了 macOS 下的优秀开源软件的列表。
      并且需要注意的是，macOS 还拥有着非常多的优秀商业软件，选择起来还是要看每个人的需求。
      当然了，如果你有各种功能需求而找不到合适的软件，还可以在新浪微博，发一条微博 @小众软件，就可以了，会有很多热心同学帮忙解答。

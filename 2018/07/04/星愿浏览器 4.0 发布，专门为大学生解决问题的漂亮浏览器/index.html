星愿浏览器 4.0 发布，专门为大学生解决问题的漂亮浏览器

酷软推荐 小众软件 2018/07/04

星愿浏览器是一款年轻的、基于 Chromium 的 Windows 浏览器，主要定位于大学生群体，最近发布了 4.0 版本，主打漂亮的新界面，以及视频去广告、百度盘高速下载、漫画阅读模式、直接安装 Chrome 扩展等功能。@Appinn

星愿浏览器 4.0 版本的新界面有着这样一个好基友间的故事：

星愿 4.0 版本，延续了 3.0 风格，传闻新版浏览器的UI和交互，是由 UI中国 的 CEO 董景博亲自操刀设计的，笔者联系了 UI中国 官方，验证了此事。笔者从董景博处了解到，作为设计师，他对浏览器情有独钟，一直有想为设计师群体开发一款的浏览器的念头。去年，他经朋友推荐，使用了星愿浏览器，很惊讶，这款浏览器的开发者，居然是自己的老同事（原来他和星愿浏览器的技术合伙人甘恒，2010 年在遨游同事的时候，一起合作开发过经典的遨游浏览器 2 和 3），多年的情结再次萦绕心头，主动联系上之后，一来二去，眉来眼去，就有了现在的星愿浏览器 4.0。

McAfee 黑市调查大发现：10美元就能获得机场的 RDP 访问权限

科技 雷锋网 2018/07/15

如果给你10美元，够干些什么？

emmmmm……..
够买张电影票或吃份双人堡套餐，够从国贸打车到中关村……..

那如果给黑客10美元呢？
McAfee 的安全研究员告诉我们，这足够让黑客们在暗网购买武器，攻入机场的远程控制系统，从而肆无忌惮的发送垃圾邮件、创建虚假安全警报、窃取数据和凭据、用服务器来挖矿……
据外媒 securityaffairs 报道，最近 McAfee 的安全研究员分析了一波各行业的 RDP 访问权限在黑市上的售卖情况，他们逛了几个提供此类服务的暗网商店↓↓↓

雷锋网发现，最大的一个暗网商店名为“UAS”（Ultimate Anonymity Service ），直译过来就是“终极匿名服务”，这家商店来自俄罗斯，它可以提供 40000 多个各行各业的 RDP 访问权限。
排名第二的是一家叫“BlackPass”的商店，这家暗网商店虽然 RDP 权限只有10000多个，但胜在服务种类比较丰富，它还为黑客提供其他种类的武器。
排名第三的是一家叫 Flyded 的商店，老四 xDedic 虽然只有1000，但由于2016年6月 曾被卡巴斯基的专家发现，所以名气也不小。
为什么暗网的商家们纷纷开始做 RDP 权限的生意？
因为这个工具对于黑客来说，物美价廉。

在黑客圈子中，这个工具越来越流行，尤其是用它可以来分发恶意软件，比如臭名昭著 SamSam 勒索软件有很多正是通过这个工具进行分发，更为重要的是，这种工具非常的便宜，据 McAfee 的安全研究员透露，一些高价值网络的 RDP 权限，在暗网的售价竟然不到 1 美元，黑客买来之后，有时候只是悄悄的扫描重要的信息，然后再悄悄的离开~~~
目前，黑市上的卖家提供对各种系统的RDP访问，范围从 Windows XP 到 Windows 10 的系统都有涵盖，其中 Windows 2008 和 2012 Server 分别是最受欢迎系统，分别有11000 和 6500 个待售。
RDP访问因权限的不同价格也不一样，比如最低的普通访问权限是3美元，而要得到管理员访问权限，则需要19美元。
安全研究员发现，UAS Shop 和 BlackPass 提供的服务的特点是数百个与荷兰的市政当局、住房协会和医疗机构相关的配置相同的设备系统，这些政府系统是黑客的重要目标。
在分析UAS商店时，研究人员发现最近添加的 Windows Server 2008 R2 标准机器仅售10美元，位于美国的一个主要国际机场。
卖方提供了三个不同权限，分别是用户帐户、管理员帐户以及第三方公司账号，这些公司账户包括一家专门从事机场安全和楼宇自动化的公司，以及另一个专门针对机场的摄像头监控和视频分析的公司。
这种访问可能非常危险，试想，如果黑客能控制整个机场的楼宇控制和摄像头控制，会发生什么。
雷锋网(公众号：雷锋网)发现，更研究人员感到更为震惊的是，利用这些权限，他们还可以通过相关的账号进入“机场的自动交通系统“，直接连接终端的客运系统。

现在我们知道像 SamSam 这样的勒索病毒，攻击者确实可以使用暗网售卖的 RDP 权限来实现攻击，用低廉的价格获取潜在的高价值勒索软件受害者。我们发现，只需10美元就可以购买与主要国际机场相关的系统 。
换句话说，黑客不需要高价购买零日漏洞，也不需要精心设计网络钓鱼活动或水坑攻击，直接买远程控制系统权限即可。
“政府和组织每年花费数十亿美元来保护我们信任的计算机系统。但即使是最先进的解决方案也无法在后门打开或仅带有简单的防护时提供安全保障。“

雷锋网(公众号：雷锋网)新智驾按：说起瑞典这个国家，恐怕大多数人的第一反应就是宜家。这个北欧国家在设计上确实有自己的一套，沃尔沃的雷神之锤等设计元素也早已成了汽车行业的经典。现在，瑞典的设计师们又开始定义未来了，本土自动驾驶卡车新创公司 Einride 本周在古德伍德速度节上推出了第 2 款原型车 T-Log，与第一款产品一样，T-Log 也是个超级抓眼球的家伙。

卡车终极形态？

既然要讲未来的卡车形态，那我们就先从 Einride 公司首款产品 T-Pod 说起。这辆卡车完全采用了颠覆性设计，它不但没有方向盘和油门、制动踏板，激进的 Einride 甚至直接取消了挡风玻璃和驾驶舱，整个车就像是架在轮子上的大货厢。

*T-pod
T-pod 也顺应潮流用上了纯电动设计，没了驾驶舱的它车身比一般的大货车短了不少，仅有 23 英尺（约合 7 米）长。Einride 给出的数据显示，T-pod 满载重量为 20 吨（想想那一堆电池，说明这辆车载重能力一般），除了全自动驾驶，该车还能由人类操作员进行远程操控。
Einride 的野心可不小，它们准备在 2020 年推出一套完整的交通运输系统，其中包含 200 辆 T-pods 和配套的充电设施。一旦投入使用，这套系统一年能运输 200 万个标准托盘的货物。
与 T-pod 类似，T-Log 也是一台电动自动驾驶卡车，T-pod 身上那些颠覆性元素在它身上一个都不能少。不过，T-Log 看起来更加狂野，是专为巨型伐木场而生的。性能方面，T-Log 比自己的前一代要强上不少，它一次能拉走 15 吨货物。

*T-log
不过，这样的介绍也让人生疑，这家伙不是撤掉了驾驶室吗？现在的自动驾驶在没有任何标线和指示牌的林场里能吃得开吗？各种泥土和砂砾可不是闹着玩的。
众所周知，即使是被各种传感器、地图数据和深度学习技术武装到牙齿，自动驾驶汽车的活动范围也会受到一定限制，那些可预测的环境才是它们当下的归宿。
不过，对 Einride 来说这个定律并不适用。在自动驾驶上，这家新创公司用了英伟达的自动驾驶软件，它们的目标是 Level 4。至于远程操控技术，它们则选择了 Phantom Auto 的方案。有了这项技术，遇见那些复杂的越野环境就方便多了。
Einride 表示，T-Log 搭载的 300kWh 电池充满电能行驶 120 英里（约合 193 千米）。未来两年内这款车还会不断改进，2020 年正式上路后它可能会长得更加科幻。
“斯堪的纳维亚半岛的国家伐木产业是重头，整个交通量中 20% 都与伐木产业有关，而电动车能大幅减少碳排放。” Einride CEO Robert Falck 说道。“我们想证明电动车也能经受住最艰苦的工作。”
随着各国排放政策的收紧，电动卡车的时代确实不远了。Elon Musk 就曾表示，如果要跑 100 英里的短途运输，特斯拉 Semi 卡车每英里运营成本才 1.26 美元，而传统柴油卡车就得 1.51 美元了。据雷锋网新智驾了解，西门子已经开始在德国、瑞典和洛杉矶港试用电动卡车了，不过驱动这些车的不是车载电池，而是头上的电线。除此之外，卡车市场还有一家名为 Nikola One 的公司，不过它们用的是氢燃料电池，无需受充电时间长的困扰。

关于自动驾驶卡车发展的争论


虽然自动驾驶卡车市场已经进入了百家争鸣的大发展时期，但想让它们真正落地，恐怕要解决的问题还很多。
一直以来，科技巨头和风投机构都对自动驾驶卡车信心十足。在其眼中，最多 10 年，自动驾驶卡车就会成为人人熟悉的产品，这一巨变将为整个运输行业省下 3000 亿美元的成本。
反对的声音来得更为猛烈。
卡车司机和分析师们却显得很淡定，他们认为自动驾驶卡车只是海市蜃楼。
物流巨头 Zipline Logistics 创始人 Andrew Lynch 就表示：“我认为自动驾驶卡车技术离成熟还有很长一段距离呢，现在说准备好都有些过于乐观。”
交通分析公司 Broughton Capital 也相信，自动驾驶卡车未来 5 年甚至连高速公路都统治不了。
除了技术上的限制，自动驾驶卡车还得过法律这一关，如果除了事故到底如何划分责任又由谁来赔偿呢？
而且，大量卡车司机丢掉工作也有可能造成社会动荡。就拿美国来说，有 440 万个工作岗位都与驾驶有关，其中卡车司机就占了 250 万个。
雷锋网新智驾查阅最近的研究显示，2030 年自动驾驶卡车可能会减少美国和欧洲 50%-70% 的驾驶员需求。也就是说，两个地区 640 万名专业司机，可能会有 440 万名丢掉饭碗，恐怕没有哪个政府愿意面对如此汹涌的失业人口吧。

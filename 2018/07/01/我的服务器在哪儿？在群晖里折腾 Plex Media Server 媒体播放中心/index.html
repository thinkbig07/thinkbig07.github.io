我的服务器在哪儿？在群晖里折腾 Plex Media Server 媒体播放中心

酷软推荐 小众软件 2018/07/01

这个 B 装大了。
起因是这样的，青小蛙在开箱了群晖 DS218+，加内存 后，给 DSM 的访问界面加上了带有 HTTPS 的域名，这样就能在浏览器里看到小绿条了，但是，没想到居然栽在了 Plex Media Server 上。@Appinn

Plex 是一套媒体播放器及媒体服务器软件，让用户整理在设备上的有声书、音乐、播客、图片和视频文件，以供流至移动设备、智能电视和电子媒体播放器上。
Plex 可用于 Windows、Android、Linux、OS X和FreeBSD。
另外，Plex 亦让用户通过该平台观看来自 YouTube、Vimeo 和 TED 等内容提供商的视频。Plex 亦与 Bitcasa、Box 和 Dropbox 等云服务兼容。via 维基百科
这是很奇怪的事情，因为上次青小蛙折腾 Plex 的时候也遇到了这个问题，最终是放弃了。而这次，因为好友的 Plex 是好的，越发觉得奇怪。
搜遍全网，只看到一个说法：Plex 默认禁用远程访问，所以…怎么办？
无意中，青小蛙突然使用了 IP 地址访问 DSM，然后打开了 Plex Media Server 界面，神奇的事情出来了，服务器出现了…
呃，真的就这样简单，只需要使用 IP 地址访问 Plex 一次，就可以扫描并添加你自己的服务器到 Plex 了。

而在正式用上 Plex Media Server 之后，我就卸载了群晖自带那个 Video Station 和媒体服务器两个套件。
Plex 的安装也异常简单，在 DSM 的套件中心里，直接点击安装按钮就行了，第一次打开的时候记得用 IP，就可以设置 Plex Media Server 和添加多媒体文件了。
之后，每个平台基本上都有客户端原生支持 Plex，比如 Kodi、Infuse 等等，而 Plex 另外一个特别之处在于可以将自己的媒体库分享给好友。
这个就厉害了。
比如：

你可以与好友分享彼此的媒体库，当然想要比较流畅的播放，上行宽带越宽越好，一般 10～20M 就可以了。
而在分享的时候注意一下，如果你有一些奇怪的动作片，那么记得使用 Plex 的家庭账号功能，而这个功能是付费的，所以，另外一个解决方案是再注册一个账号，然后将合规的内容分享给新的账号就可以了。
嗯，就是这样。
如果你还有其他 Plex 的问题，欢迎留言讨论。

Solid Explorer – 全能 Android 文件管理器

酷软推荐 小众软件 2018/07/02

Solid Explorer 是一款非常优秀的 Android 文件管理器，Material Design 设计风格，双栏布局，可拖拽操作、支持 ROOT 权限、多媒体浏览器、压缩包支持，Chromecast 流支持等众多功能。@Appinn

其实 Solid Explorer 在青小蛙的手机里有 2 年的时间了，之所以一直没有介绍是因为 Solid Explorer 并没有免费版本，只提供了 14 天试用版。
而更早在 2013 年，就有同学推荐了 Solid Explorer：

@ff 说到：Solid explorer 比 ES 好用多了，双栏布局，操作方便，同样自带磁盘分析器，而且 Solid explorer 跟这个 DiskUsage 可以互相调用，DiskUsage 比那些管理器自带直观多了
同样是那篇文章之下，对于 Solid 的讨论都超过了 DiskUsage 😂

再来说说 Solid Explorer 的一些功能吧，青小蛙最常用的是「应用」功能，它能够列出所有系统应用与用户应用，并且可以轻松将应用的 apk 文件分享出来。
很多次青小蛙从 Google Play 安装后的软件，在第三方镜像站点无法下载，或者软件自身已经下架，都是通过 Solid Explorer 导出来的。

每次导出的文件名为 base.apk，可以直接拿来安装。
上看到 Solid 的书签功能，青小蛙一直把截图放在书签里面，多张截图的时候快速定位非常有效。
其他方面，Solid Explorer 还能连接各种云服务，比如：

  • Google Drive
  • Dropbox
  • OneDrive
  • LAN / SMB
  • FTP
  • SFTP
  • WebDav
  • Yandex
  • Owncloud
  • Box
  • SugarSync
  • MediaFire
  • 插件支持更多
    没错，Solid 可以通过插件扩展更多的功能， 青小蛙了解到的插件包括以下一些：
  • FTP 服务器
  • AWS S3
  • 各种图标套图
  • Mega 插件
  • 等等
    由于国内各网盘的开放程度，以及不确定程度，目前可用的只有坚果云的 WebDav 方式。而自建云服务，比如群晖，是可以实现 LAN / SMB 以及 Owncloud、nextcloud、seafile 等等。
    而在连接了网盘或者云服务之后，如何传输文件，Solid 也帮你搞定了：横屏双栏 – 拖拽。
    只需要将屏幕横过来，Solid 会自动打开双屏，这样就可以方便的在左右栏传输文件了，来看视频，双屏的操作在大概 16 秒钟：
    如果你的设备拥有 root 权限，那么还可以打开手机的 根目录，查看到所有文件夹。
    以及，由于这是一款 14 天试用版本，青小蛙就只提供 Google Play 商店地址了。

装修还是不能任性，本想搞「极简风」，结果满屋「廉价感」

今日热点 知乎日报 2018/07/16

你应该做做减法了，再增加摆件只会更乱。现在的问题不是廉价，而是又局促又压抑。
每个人都有自己喜欢的风格，没有必要非得弄成现在流行的所谓北欧极简风，买一堆网红家具，千“家”一面看多了也厌。但是，你喜欢的风格，我姑且概括一下，第一张是欧美豪宅（？）风，第二张是中式（？）土豪风，审美怎么样先不论，这两种风格都需要足够大的房子才能撑起来。



我发现一个蛮有意思的现象，日本设计师写的书，多是教人怎么利用空间，美国设计师写的书，有一特高频的词，怎样“填满”空间。所以装修的风格取决于你房子的空间，有的房子卧室大得能打球，放一张细腿铁架床就不好看，有种孤苦伶仃家徒四壁的感觉。像你的情况，这个卧室本来就不大，要承担睡觉（床），化妆 / 工作 / 学习（梳妆台），休闲（贵妃榻），书房（书架）这么多功能，实在是不堪重荷。就说那个梳妆台，你真的坐的进去么，是不是经常磕到腿？坐下之后是不是不敢乱动，一动就碰到后面的贵妃榻？打扫卫生的时候是不是各种拐弯抹角清理不到？
另一方面，你喜欢的风格，第一张图是深蓝色主色调，第二张是深木色。这种深色非常不好用，需要家里采光非常好，且严格控制使用面积，才能达到好的效果。第一张图里，床上用了一些白色，墙上挂了几个青花瓷的不知道什么东西，都是为了提亮。第二张也是，沙发和墙面颜色都不深，顶是白的，还摆了玻璃瓶的绿植和黄铜的台灯。而且这两个房间，一看这个顶天立地的窗帘，就知道窗户很大采光很好，所以才敢用深色。
深色用的好，确实非常提升质感，但问题是，你的卧室看上去采光并不好。白天开着灯拍照都能拍成这样，再大面积的用深色（床头加床单），估计白天一出家门，有种“洞中方一日，世上已千年”的感觉吧。
综上，你真的没办法任性由着自己喜好来装，房子的面积和采光都不允许。倒是你贴的第三张，跑偏了的极简风，可以借鉴一下。
你（家）装修的宗旨应该是，让房间显大显亮。装修的时候把握一些基本原则，最后出来效果通常都不错，也许没有亮点，但至少不会有什么雷点。

  1. 多用浅色少用深色。浅色能大幅度提升家里的明亮程度。你选的第三张图，清一水的白色，“雪洞一般”。好不好看先不说，起码很亮堂。
    非要用深色的话，千万千万控制面积，小范围画龙点睛就行了。像你房间这种采光，用一个沙发的面积我都觉得太大了，一两个靠垫就行了。
  2. 多用纯色少用碎花。纯色比条纹整齐，条纹比格子整齐，格子比大花整齐，大花比碎花整齐，粉色碎花 + 绿叶是灾难（我妈的最爱，家里哪哪都是）。 慎用格子，因为格子的大小，线条的粗细，不太容易把握，选的不好分分钟打回八十年代。
    只用纯色容易显得呆板，白茫茫一片大地不知道视线该落在哪，也不好。可以搭配条纹或几何图形，稍微活泼一点，或者考虑装饰品（是的，这种时候再考虑装饰品），挂画、绿植、摆件、香薰蜡烛都可以。黄铜和玫瑰金是宝贝，谁用谁知道。慎用鲜花 / 假花，挑不好容易花红柳绿。实在喜欢粉色，考虑下脏粉配浅灰，原理见下一条。

  3. 多用低饱和色少用高饱和色。简单粗暴的说就是少用浓墨重彩的艳色，掺点“高级灰”，多丑的颜色都能救回来。传说中的莫兰迪色系，粉 + 绿 + 黄 + 蓝放一起，饱和度低一点也挺和谐的。

    你的房间里，紫色的床头，蓝色的床单，红色的假花，粉色的挂画，橙色的靠垫，高饱和度的颜色太多了。实在喜欢艳色，小面积用一个就行了，最多两个，不过那要考验你撞色的能力了，慎用。

    4 多用直线少用曲线。视觉上直线比曲线整洁，方形比圆形开阔。美式风格喜欢用曲线，比较容易填满空间。而你放的第三张图，除了一言难尽的吊顶，一条曲线都没有，非常干练整齐，用在厨房这种容易乱的地方，效果通常不错。而且就算是直线，也要尽可能减少线条，因为线条越多，看上去物体就越多。我们之所以需要收纳，就是为了把一堆物品各自的线条，变成收纳盒一个面四条线，视觉上整齐多了。
    而你的房间，床头是椭圆的，还有灾难一般的浮雕。梳妆镜是圆的，浮雕也不少。墙上有个圆形的钟表（？），地上有个“婀娜多姿”的花架，贵妃榻有条不规则曲线。桌子椅子虽然是方的，但八条腿没闲着，又有浮雕。唯一方形的画框，上面竟然还是浮雕！整个房间数不清有多少线条，第一眼都不知道看哪。其实你摆在外面的东西很少，收纳应该做的不错。房间看着这么 busy，全靠家具的“功劳”，白瞎了你辛苦收拾。
  4. 多用轻盈的家具少用臃肿的家具。同样的功能同样的面积，轻盈纤细的家具比一坨蹲在地上的家具，视觉上轻松太多。简单粗暴点说，四条小细腿的家具容易显得空间大。现在的家具质量基本都过关，没必要太纠结家具结不结实，有几个人真能把沙发坐塌了不成。
    暂时就想到这么多。当然，不是说深色，花纹，艳色，曲线，圆形跟臃肿的家具不能用，这些用好了，气质能提升好几档。但是对像你我这样的普通消费者，没有人家设计师的金刚钻，就别自己作死去玩高级的瓷器活，弄出来辣眼睛的概率比较大，不如先老老实实的把黑白灰用好，慢慢再进阶。
    至于房间的改造，题主 20 岁，那这个应该是爸妈的房子吧，如果非常非常想改善居住环境，要不跟爸妈商量商量，重新装修一下？其实主要都是家具的锅，撕掉壁纸刷个白墙，往里填家具就行了，地板懒得弄就算了。
    如果是租的或者不想折腾，建议在原来基础上再糊一层壁纸，留下书架和床身，其他扔了。买个床头或者干脆不要床头，多堆几个靠垫看着跟床头是一样的。换窗帘，买桌椅。梳妆镜就算了，稍微近视一点离这么远画眼线根本看不见，买个专门的化妆镜就行了。如果不化妆不在卧室工作学习，桌子其实也没必要买。善用马云爸爸，其实这几样都不贵。
    当然我也不是专业的，说的也不一定对，你就随便一看。具体的还要多看看装修帖和讲设计的书，真打算装修，千万千万看够海量装修图，明确自己想要什么再动手。

Notes by Firefox – 火狐推出「安全便签」应用 [Firefox / Android]

酷软推荐 小众软件 2018/07/12

Notes by Firefox 是 Mozilla 专门为 Firefox 浏览器推出的便签笔记功能，目前已经发布了专属的 Android 版本，可以脱离 Firefox 浏览器单独使用，通过 Firefox 账号同步，基于端到端加密。@Appinn

在 Firefox 上面的 Notes 位于边栏，让青小蛙想起了曾经的 Opera…
在桌面版的 Firefox 中，非常方便于使用，不过还需要在这里安装一下才可以使用。

而在 Android 手机中，就需要安装 Notes by Firefox 了，目前的 Firefox for Android 还不支持直接使用 Notes。
Notes by Firefox 长这个样子：

目前的同步功能在国内还处于正常可使用范围。需要登录 Firefox 账号。
另外，由于没有 Web 版本，适合于使用 Firefox 的同学，如果使用其他浏览器，在桌面端似乎没有太好用的同步工具。
由于太简单了，所以更像是便签，而非笔记 😂

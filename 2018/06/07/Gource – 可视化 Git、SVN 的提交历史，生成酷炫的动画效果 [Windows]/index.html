Gource – 可视化 Git、SVN 的提交历史，生成酷炫的动画效果 [Windows]

酷软推荐 小众软件 2018/06/07

Gource 是一款非常有趣工具，它能够为 Git、SVN 的提交历史生成炫酷的动画效果，能够显示出参与作者、分支、合并等等。@Appinn

感谢 @ruanyf 的推荐：

Gource 是一个很好玩的可视化工具，可以将代码仓库的历史变成视频，支持 Git 和 SVN 等多种格式。
Gource 能显示的信息很丰富，来看官方介绍视频吧
用法也简单，先准备 git 项目路径，比如 c:\git\abc\
在安装来 Git 的 Windows 电脑上，打开 cmd，进入 gource 目录，然后运行：

1gource.exe c:\\git\\abc\\
然后就能看到视频了，如果想保存为视频文件，则使用参数 -o 即可。
另外，最新的 Gource 0.48 提供了源码，青小蛙在 Ubuntu 上编译缺了好多依赖装不过来，最终放弃了…
感兴趣的同学可以前往 Gource 官方了解。
原文请参见: http://www.appinn.com/gource/

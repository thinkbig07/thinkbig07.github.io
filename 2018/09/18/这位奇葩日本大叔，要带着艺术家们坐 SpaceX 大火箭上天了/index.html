这位奇葩日本大叔，要带着艺术家们坐 SpaceX 大火箭上天了

今日热点 知乎日报 2018/09/18

机会难得，要不咱们来八一八前泽大叔和他的 ZOZOTOWN ？
先聊点正经的（再聊不正经的），ZOZOTOWN 可以说是日本当今萎靡的互联网业界中唯一一支独秀了。
在日本有一个词叫做\”兆元企业\”，意思就是年销售额可以达到一兆日元的企业。和我国国情不同，这个兆元的队伍里大多都是传统企业，比如丰田、日本邮政、丸红商社、花王，略微和 IT 沾边的只有 NTT Data，Recruit 和 ZOZOTOWN。

在乐天和雅虎稳稳占领市场的情况下，ZOZOTOWN 异军突起，一跃成为日本最大的服装贩卖网站。你可以想象一下，在淘宝大行天下的时代，有一位年轻人一手创立了一家服装 EC，并且最终占领宝座的场景。

用户体验、商品、创新性都没的说，完爆日本一般的互联网应用（包括乐天雅虎），之前还推出了脑洞大开的 ZOZOSUITS。
关于 ZOZOSUITS，可以移步最近风靡日本的测身体秋衣秋裤，让我知道了日本女孩的平均胸围。
ZOZOTOWN 的公司其实叫做「START TODAY」，不讲诸多画风奇特，就只说一点吧。在加班成风热爱劳动的日本，在 996 的互联网界，这家公司的规定工作时间是早晨九点至下午三点，六小时工作制。
然而比起前泽友作本人，ZOZOTOWN 的光芒就逊色的多了。
75 年生人，高中学历，当做摇滚歌手出过唱片，现在，日本人称他为『最爱购物的男人』。

不光 100 亿的豪宅，他还经常在 SNS 上晒自己的豪机（据知情人的信息，这样一架飞机一年的维护就要 3 亿日元）、豪车，不是日本第一首富，却完完全全是日本第一\”壕\”。


因为喜欢艺术，所以他也是拍卖会的常客，画作和艺术品、古董经常是几百亿几百亿的买。
之所以不能说前泽是日本的 Elon Mask，是因为他的简历实在不能称得上是典型的社会精英和知识分子。
前泽高中差点没毕业，最后靠着勉强合格的出勤和分数混到了高中文凭。毕业后靠着高中时代打工的钱去美国进行了半年的音乐游学，之后在与朋友一起组建的乐队\”SWITCH STYLE\”担任鼓手。


2001 年，或许是感觉到了互联网的\”Big Wave\”，前泽毅然停止了乐队活动决心投身互联网。只有高中毕业的他去秋叶原购买了网站搭建的书籍，通过自学设计了编写了一个 EC 网站。这就是当今 ZOZOTOWN 的原型。
前泽不是一位典型的商人，而是一位典型的\”热情驱动\”企业家。无论是员工还是朋友，都表示他的脑洞异常的大，经常会想到常人思维以外的点。比如 ZOZOTOWN 十周年时，他与员工秘密策划了一个『商品 0 元』的活动。到了活动那天，衣服突然毫无预兆的价格变为 0 元，用户们都惊呆了，以为是什么系统错误，于是在推特上四处通知『ZOZO 网站好像错误了，现在衣服都免费买！赶紧买！！』
自然，这样的企划也为 ZOZOTOWN 带来了巨大的流量。而你所不知的是，在日本，大幅度的全面打折活动从来都不是商家常用的手段，很多零售企业最出血也就打个全场 85 折。
除了 ZOZOTOWN 以及前泽最近又买了什么，日本人民最关心的就是前泽最近的绯闻女友又是谁了。
说到他的现任女友，也算和咱们中国人民是老朋友了呢。

没错，就是日本著名女演员刚力彩芽。
还有一位绯闻前女友是纱荣子。

同样是绯闻前女友，女演员下京庆子。

看来前泽大叔比较喜欢女演员。
传说前泽与纱荣子分手之后，因为想要女友，特地还搞了个『女友试镜』，还真有人报名参加。
也有传言说刚力彩芽是从下京手中作为第三者把前泽抢到了手……还有说他有三个孩子，都是不同的母亲……众说纷纭，反而让前泽友作的知名度节节高升，不断为日本人民提供茶余饭后的各种大瓜。
前泽友作有一句名言，就是他总是『挣多少花多少』。
一个亿万富翁，居然有时候吃饭还要看钱够不够点套餐。
这样的前泽友作，成为第一个登月旅客，也算是意料之中了。更有趣的是，前泽说希望邀请画家、雕塑家、电影导演、时装设计师、音乐家和其他艺术家一同登月。他希望通过登月的体验，激发艺术家们的灵感，创作出更伟大的作品。
这个项目被命名为＃dearMoon。是作为现代艺术振兴财团会长、从小喜爱月亮的前泽在这个庸碌的社会中，为人类打造的浪漫。
【我要去月亮了，和艺术家们一起。】

我想这会是他最大，也最有意义的一次购物吧。

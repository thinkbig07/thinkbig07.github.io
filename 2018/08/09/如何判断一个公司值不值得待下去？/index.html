如何判断一个公司值不值得待下去？

今日热点 知乎日报 2018/08/09

要判断一家公司值不值得待下去，需要从公司基本面和个人职业价值观两个维度来综合分析。
先说个结论：公司基本面良好，个人职业价值观与公司匹配，就值得待下去。
那到底公司基本面怎么分析，个人职业价值观是什么东东，咱们下面展开聊。
一. 公司基本面
公司基本面包括以下内容：

  • 公司是否创造正向的价值
  • 公司所处行业（领域）的状况
  • 公司在行业内的地位
  • 公司的经营状况
    一个一个来说下。
    1. 公司是否创造正向的价值
    我们可以通过问自己一个关键问题——“如果这家公司倒闭了，会对社会产生什么影响？”——来判断一家公司是否创造正向的价值。
    如果答案是“没什么影响”或者“少一批人上当受骗”，那这家公司就不创造正向价值。
    如果答案是类似“大家就买不到好用的产品”、“大家做某类正当的事情将很不方便”这种，那这家公司就创造正向价值。
    现在，有两个公司，C1 的业务是短信诈骗，C2 的业务是英语培训。
    请设想你是 C1 和 C2 的员工，问自己“如果这家公司倒闭了，会对社会产生什么影响？”来判断它们值不值得你待下来。
    2. 公司所处行业的状况
    要选择处于上升期、成熟期的行业。
    夕阳行业的公司不要待。
    至于行业是否向好，可以看各行各业的数据分析、国家规划等信息来判断。
    3. 公司在行业内的地位
    如果公司在行业位置靠前，或者产品独特在行业内细分市场举足轻重，那这样的公司，待下去往往是没不错的。
    如果公司在行业内垫底，也没什么特别的产品，大概率不值得待下去。
    4. 公司的经营状况
    公司有无主营产品（服务），营收如何，利润如何，负债如何，融资如何，这些都需要了解。假如一家公司没什么朝不保夕，负债累累，工资都发布出来了，那基本上就没必要待下去了。
    二. 个人职业价值观
    所谓个人职业价值观，就是你认为在职业中，什么最重要。比如你认为“工资高福利好”最重要，那它就是你的职业价值观；比如我认为“社会需要与贡献大”很重要，那它就是我的职业价值观。
    常见的职业价值观有九种：
  1. 工资高福利好
  2. 工作稳定有保障
  3. 人际关系良好
  4. 工作环境舒适
  5. 能充分发挥自己的能力和专长
  6. 有较高的社会地位
  7. 社会需要与贡献大
  8. 工作压力小
  9. 有受教育的机会
    持有不同价值观的人，选择不同的公司。
    张三认为“工资高福利好”就是一切，其他都可以不管不顾，那假如前面提到的 C1 公司，即主营短信诈骗的公司，给出的薪水很高，他就可能会选择待下去。
    李四认为“有较高的社会地位”是最关键的，那他可能就会选择在国家企事业单位待下去。
    所以，我们需要明确自己的职业价值观，找到我们在职场中最看重的特质，然后才能判断自己要不要在一家公司待下去。
    三. 判断去留的逻辑
    有了对公司基本面和个人职业价值观的分析，我们就可以明确判断一家公司是否值得待下去的逻辑了：
    1) 如果公司基本面很差，那就没必要待下去。
    2) 如果公司基本面良好，接下来看再看个人职业价值观，如果与公司匹配，那这样的公司就值得待下去；如果与公司不匹配，那也没必要待下去。

    如何判断一个公司值不值得待下去？
    三个维度，可以比较方便的判断公司的潜力：
    一、看战略（经营）
    方向正确，犯点小错误也没关系，方向不对，随便一点问题都很致命。这是一艘起飞的火箭，还是一艘将要沉没的大船。
    现在市场很成熟了，信息非常流畅的传递，如果战略方向正确，你会发现资源会快速向公司聚集，更多的客户，更高的牛人，更快的融资。反过来说就是只做事，没反应，辛辛苦苦设计的产品，没有客户买单。
    如果这些都看不到，就看老板和股东对公司的态度，如果公司很有希望，老板一般是最拼的，因为他是第一受益人。如果老板一周都来不了几次，随便扔给经理人去管，那么这公司肯定没什么未来。
    二、看效率（管理）
    客户的需求，设计的想法，质量的异常，从出现到解决需要多久。一家效率极高的公司，即使做不到最好也绝对饿不着。
    有些公司规模虽然不大，运转效率极高，能不能赚钱，能赚多少钱，一清二楚。
    有些公司管理水平太烂，运转效率极低，明明是好产品，好员工，就是连问题出在哪都搞不清楚。
    三、看团队（人）
    老板、中层、同事是不是牛人，聪明人方向不对改方向，管理不行学管理。有些老板虽然没有什么学历，但是对业务核心的把握非常精准。
    人如果不行，有什么资源都白扯。看人行不行也很简单，就是看他懂不懂行。客户是谁、客户的需求是什么、我们的产品能满足客户需求么，我们的竞品有哪些，和我们之间有何优劣、公司有哪些资源和能力，公司有什么短板和不足，能力能增强么，短板能补足么。
    这些说的不一定对，但是如果连想都没想过，能把公司干好，也挺难的。
    最后提醒一句，人和人也有相性，即使公司一般，但是老板特别赏识你，上司特别愿意带你，同事非常支持你，对别人可能不值得，对你还是值得花点时间和精力好好干一干。

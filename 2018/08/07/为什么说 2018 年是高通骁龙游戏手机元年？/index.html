为什么说 2018 年是高通骁龙游戏手机元年？

科技 雷锋网 2018/08/07

对于众多游戏从业者来说，举行于 8 月 3 日到 5 日的 2018 年第十六届中国国际数码互动娱乐展览会（ChinaJoy）是绝对不可错过的重要舞台；从 IP 方、研发商、发行商、渠道商到设备商的整个游戏产业链，无不参与其中。而今年，ChinaJoy 更是首次迎来了一位超重量级的参与者——高通骁龙，它与近二十家合作伙伴一起，为玩家们带来了移动游戏领域的前沿创新科技。
实际上，在高通的定义中，2018 年是骁龙游戏手机元年。

手游时代的全面到来

在过去的几十年里，游戏产业经历了巨大的发展，而游戏设备的产品形态也随之发生了一系列改变；无论是街机游戏、PC 游戏和家庭游戏系统，还是掌上游戏机、简单手机游戏乃至于便携式的游戏主机（比如任天堂 Switch），都在游戏发展过程中扮演了重要角色。
然而从 2011 年开始，全球智能手机市场开始强势增长，在销量和用户呈井喷式爆发的同时，智能手机的硬件本身更是快速迭代；发展至今，智能手机上也已经能实现媲美 PC 或游戏主机品质的游戏体验。
当然，在此过程中，以智能手机游戏为主体的移动游戏市场也越来越大。

根据市场分析公司 Newzoo 在今年 5 月份发布的最新版《全球游戏市场报告》，移动游戏市场同比增长 25.5%，达到 703 亿美元，这意味着移动游戏营收在强劲的增长中占比首次超过 50%；其中，来自智能手机游戏的营收占据整个移动游戏市场的 80%，达到 564 亿美元。
当然，亚太地区尤其是中国的手游市场更是一片火热。报告显示，亚太游戏市场规模今年预估较去年增长 16.8%，基本上要归功于手游的增长。同样，中国游戏市场最主要的驱动力也是手游，而中国的手游市场在 2018 年将会占总体游戏市场规模 61% 的份额。
可见，中国游戏产业的发展已经全面进入到手游时代。
正是基于这样的大背景，在中国智能手机市场深耕多年的高通，也将手游作为其移动技术创新的着力点。雷锋网了解到，作为移动行业的领军企业，高通一直致力于打造顶级的移动游戏体验，并为此做了巨大投入；2017 年骁龙移动平台就率先支持了数款主流手游的高帧率模式，让移动终端的游戏玩家享受到了更稳定、流畅并具有更锐利画质细节的体验。

骁龙 845 的极致游戏体验

到了 2018 年，高通更是联合小米、一加、OPPO、vivo、锤子、黑鲨、华硕、三星等业界领先的手机厂商陆续推出了多款搭载骁龙 845 移动平台的顶级移动终端，为玩家们提供了更多各具特色的产品选择，让他们能尽情享受酣畅淋漓的手游体验。除此之外，高通还首次亮相 ChinaJoy，为玩家们亲自展示了骁龙 845 平台在手游方面的极致体验。
那么，骁龙 845 是如何在游戏方面发挥强大性能的？
从硬件的角度，骁龙 845 搭载了 Adreno GPU、Kryo CPU、Hexagon DSP 以及 LTE 调制解调器、Aqstic 音频芯片等核心硬件单元，它们结合起来为用户带来了囊括游戏图形处理、高能效、网络连接、音频等打造顶级游戏体验的整体解决方案。

其中 GPU 作为游戏的图形处理核心，高通在骁龙 845 中将其升级为 Adreno 630 视觉处理子系统，带来了更快、更高效的图形处理。它不仅对上一代 GPU 有 30% 的图形处理速度和能效提升，同时在显示接口数据吞吐带宽增加达到了 2.5 倍。同时，Adreno 630 还支持包括 Vulkan 在内的最新的图形和计算 API，并在处理效率、内存管理和通用 GPU 性能方面都有提升。
得益于这样的图形系统设计，Adreno 630 相对于其他同类竞争对手有着诸多优势。根据 GFXBench 和 The Linley Group 的公开报告，在基于 OpenGL ES 3.0 和 OpenGL ES 3.1 的图形性能测试中，骁龙 845 比其它竞品的 GPU 在性能上要高出 30%-50%。另外，在同样的硅片面积上 Adreno 630 GPU 能输出更多性能，是整体GPU设计能力的集中体现；而 Adreno 630 的单位芯片面积能够输出的性能是其它竞品的两倍以上。可见，无论是在图形系统设计还是在架构设计能力上，Adreno 630 都领先于其他的 GPU。

当然，除了 Adreno 630，骁龙 845 的 Kryo 385 CPU 基于 Arm Cortex 技术打造，采用了最新的第二代 10nm LPP FINFET 制程工艺，性能核心和效率核心都有自己的独立 L2 缓存，同时也可以使用系统共享的 L3 缓存，单核能力会有大幅度提升，可在游戏、应用程序启动时间和重负载型程序上实现高达 25% 的性能提升；X20 LTE 调制解调器支持千兆级 LTE 连接，并支持多千兆比特 11ad Wi-Fi 及智能双通路 Wi-Fi，确保游戏流畅不掉线、无延迟；而人工智能引擎 AIE 通过顶级的异构计算则可以以最佳方式分配、科学处理终端的工作负载……这些都构成了骁龙 845 为用户提供流畅、逼真的卓越移动游戏体验的硬件基础。
在硬件之外，骁龙 845 还支持众多的工具和 API。这其中最值得强调的是骁龙 845 对下一代图形 API Vulkan 的支持，后者不仅可以提升游戏的图形和光效，也可以减低功耗和提升帧数。从目前业界的一些游戏和应用来看，目前骁龙对 Vulkan 的解决方案支持是最好的；而且根据雷锋网的了解，市面上现在已经有了一些商用的 Vulkan 游戏（比如说王者荣耀），可以直接在骁龙 845 以及骁龙 845 之前的一些移动平台上运行。

除了 Vulkan，骁龙 845 还能够支持高通 3D 音频 SDK、开发者的系统辅助工具骁龙 Profiler 和 帮助开发者最大程度的利用Adreno GPU性能的Adreno GPU SDK 等 API 和工具；简单来说，骁龙的方案给游戏开发者提供一套完整的支持，为他们在游戏使用和开发过程中提供最佳的性能与能效，实现长时间游戏场景下的强劲、稳定性能。另外，在合作方面，高通还与网易进行合作，双方基于网易的 Messiah 游戏引擎合作开发优化了多款热门游戏，比如说荒野行动等，为手游爱好者提供更加“快”“顺”“久”“酷”“智”的顶级的游戏体验。
值得一提的是，除了手机游戏方面的游戏体验，在XR领域，骁龙 845 还能够通过视觉、音频与交互上的创新技术，比如支持室内空间定位（room-scale）六自由度（6DoF）和即时定位与地图构建（SLAM）等，为玩家带来身临其境的沉浸式 XR 游戏体验。此前，高通还发布了面向XR设计的专用骁龙XR1平台，通过人工智能功能提供更佳的交互性、功耗表现和热效率，在为用户提供高品质的XR体验的同时支持OEM厂商开发主流终端。
实际上，在本次 ChinaJoy 上，高通也携带了小米、一加、OPPO、vivo 等多家合作伙伴的搭载骁龙 845 的智能手机和小米 VR 一体机等多款 VR 设备亮相展台，展示了以手机通过USB Type-C连接电视实现的4K高清游戏画面输出（4K TV-Out）、沉浸式的移动XR体验，以及骁龙845支持的其它出色特性。比如基于高通人工智能引擎 AI Engine 实现的先进AI娱乐体验——“高能舞室”是手机 QQ 上一项有趣的AI交互特性，AI Engine的加持可以帮助其提升帧率，实现更流畅的交互体验；高通还与商汤科技还共同实现了更高分辨率的 SenseAR “美体塑型”特性。

而在终端厂商vivo的展台，基于骁龙 845 的 vivo NEX旗舰版“百人团战，志竞非凡”——王者荣耀边境突围全国总决赛吸引了大量热情玩家的参与和关注；百名玩家通过搭载骁龙845移动平台的vivo NEX旗舰版智能手机，展开了酣畅激烈的“战斗”，切身感受了由骁龙845支持的“电竞级”移动游戏性能。

雷锋网总结

对于整个游戏行业而言，移动游戏已经在 2018 年成为其占据半壁江山的主体市场；而对于高通而言，伴随着骁龙 845 在移动游戏领域的卓越表现，2018 年也已经成为高通在移动游戏行业强势发力的一个重要时间节点。在雷锋网(公众号：雷锋网)看来，高通在 2018 年首次参加 ChinaJoy，也可以作为这一重要时间节点的注脚，也更反映了高通将推动游戏领域技术和体验创新的决心。

因此，如果说 2018 年是骁龙游戏手机元年，那么在接下来的发展中，高通骁龙也势必会在未来移动游戏产业的技术创新发展中扮演愈加重要的角色，我们拭目以待。

雷锋网按：在谷歌手中未能实现商业化应用的波士顿动力公司（Boston Dynamics），被软银收购一年后，如今终于对外宣布要量产其机器人军团了，波士顿动力公司创始人兼CEO Marc Raibert在6月CEBIT公布了具体商业化过程。

爆火于网络

波士顿动力公司自成立之初共带来9款多足仿生机器人，但是时至今日却一直没有一款能够实现商用。若论及其火爆原因，一方面是来自于其在网络上公布的诸多牛气十足的在机器人研发过程中的相关炫酷视频，动不动就有千万级的点击量；另一方面则是由于其历任金主爸爸都是自带流量的“主儿”：DARPA（美国国防高级研究计划局）、谷歌、软银。

但是经年之后的波士顿动力，却仍然面临着技术研发爆火，却没有一款产品能够商业化应用的局面。然而就在前不久，波士顿动力终于对外公布了其商业化应用规划，最早在2019年底可以看到相关产品面市。

商业化应用规划

波士顿动力公司创始人兼CEO Marc Raibert在CEBIT上透露，波士顿动力公司对于自己的整个商业化进程规划分为近期、中期和长期三个阶段。

近期商用规划

波士顿动力近期商用规划包括娱乐、紧急救援、安防三大应用场景，Marc Raibert认为，娱乐和紧急救援队对于波士顿动力的机器人已问题不大，但是现在还不是很看好在安防领域的应用。
娱乐：由于在娱乐方面的应用不需要做任何特殊赋能，因而也是最容易实现的，以现在波士顿动力的明星机器人的实力，其实已经可以实现这一方向的应用。
紧急救援：将机器人用于与人类协作执行紧急救援任务。对于执行紧急救援任务而言，多功能移动性很重要，波士顿动力的机器人在这方面有较为可观的表现。不过就如何适应高温、潮湿等恶劣环境这一问题仍需要进一步解决。
安防领域：虽然Marc Raibert认为安防领域是机器人该做的事情，但是并不看好其机器人在这方面的应用。由于在室内，机器人的移动性无法得到充分利用；在户外的复杂环境（例如丛林或工厂）中应用时又面临长期自主运动的问题，现在波士顿动力的机器人在这方面仍需要与人协作完成。

中期商用规划

波士顿动力中期商用规划分为仓储物流和包裹配送两大应用场景。
仓储物流：波士顿动力机器人的独特之处在于拥有优秀的平衡能力、灵活性和机动性。而在地势平坦的仓库中，加入轮子的Handle机器人将会更适用。

包裹递送：对于大范围的包裹递送，人力便宜且有效，但是机器人用于配送业务，尤其是运用到配送外卖上有两个优势：首先，配送外卖快递员会有等待时间，如果用机器人的话，等餐的这段时间的成本会更低；其次，使用机器人可以更有效地精准预测配送时间。

长期商用规划

波士顿动力长期商用规划包括施工和陪护两大应用场景，Marc Raibert明确表示更看好波士顿动力机器人在家庭陪护上的应用。
施工：建筑工地是高危环境，波士顿动力的机器人可以在建筑工地进行物料搬运，以避免人力资源的浪费，并保障人身安全，但做这样场景化应用的研发投入是否有价值，Marc Raibert表示并不能确定。
陪护：老年人和病人的陪护是机器人在未来生活中扮演的最关键角色之一。Marc Raibert认为这对波士顿动力公司长远发展来说是一个很好的研发方向， 但是家庭陪护所需的自主性操作非常复杂，可能需要十年（或更长时间）才能开发出来，因而将会是一个长期研发的过程。

Spot Mini将首先被商用

在谷歌手中未能实现商业化应用的波士顿动力公司（Boston Dynamics），被软银收购一年后，如今终于决定要量产他的机器人军团了，而其中最先被选中的是在2015年2月首发的四足机器人Spot Mini。

为什么是Spot Mini？

波士顿动力曾推出过多款双足/四足机器人，包括大狗（Big Dog）、小狗（Little Dog）、Atlas、Spot、Handle等均在业界颇具热度。其中有曾经被美国军方考虑过的大狗，也有之前在网上挑战后空翻的Atlas，这就很有必要来说一下波士顿动力为何会选Spot Mini作为首款落地产品。

早期产品定位军用，但美军并不买单

其实从波士顿动力前期推出的诸多产品的产品特性中不难发现，包括Big Dog、Little Dog、Cheetah等多款机器人都是针对复杂地形的军用场景进行研发的，这其实也与波士顿动力公司的基因有关。
据雷锋网(公众号：雷锋网)了解，波士顿动力成立于1992年，是基于麻省理工学院的Leg Lab实验室创办的。创立之初包括Big Dog、Little Dog、Cheetah等多个项目都拿到了DARPA不少的资金资助，因此我们也可以看到前几款机器人虽然在行走、避障、导航，以及防撞击、适应复杂地形上都有较好的能力，但是却在重量、外形、功耗等方面有明显不足之处，这也使得这批明星机器人在商业化道路上还有很远的路要走，一度被认为是“未来十年”后才能完成技术落地的产品，甚至最终由于产品无法面对在军用上的性能指标要求，最终也被美国军方弃用。
以Big Dog为例，这款机器人最初研发定位为战场辅助机器人，因而在波士顿动力官方给出的性能描述上也有“适应各种复杂地形的运动，并进行导航，运行速度为10公里/小时，爬坡可至35度”，实际测试中也确实具有很好的稳定性，能够快速前进和翻越障碍。但是交付到美军手中实际试用时，却由于实际应用噪音太大，并不适用于隐蔽侦查，同时由于零件复杂，野外作为出现故障后，最牛的特种兵也只能看着这宝贝疙瘩沦为一堆废铁干瞪眼。

而在之后针对特定应用场景的Cheetah、Sand Flea等均存在很大的问题，其中Cheetah虽然运动速度能够达到47公里/小时，却由于功耗巨高，使用时需要连接电缆，显然在野外作业不现实；Sand Flea侦查机器人虽然弹跳能力惊人，可以跳过9米多高的障碍物，但是一次充电只能弹跳25次。
因此，这些定位在军用场景的面容丑陋，性能“优异”的机器人在卖给军方这条路不通后，自然也很难在短期内转型到适合拿到商业市场上售卖的产品。

开启商用化道路后，面临诸多困难

推动波士顿动力从军用转向商用机器人研发的很大一部分原因来自“东家”的变更。
2013年，在安迪·鲁宾的推动下，谷歌收购了波士顿动力，同时还收购了其他八家机器人公司，一并整合进谷歌新的机器人业务部门Replicant。针对波士顿动力的收购，谷歌明确表示不希望参与国防工作，此后两家公司的防务合同开始逐步终止。也是在此后，波士顿动力在失去军方的巨额资金输血后，开始更多地考虑商业应用场景。
在谷歌麾下的日子波士顿动力依旧混地风生水起，相继推出了Spot、Atals、Handle三款机器人。

Atlas太贵，稳定性相对不足

首先我们来说一下Atlas，这款机器人是波士顿动力基于早期PETMAN双足人形机器人研发的，从外观上来看，显然已经在向大众能接受的方向走，另外，在经历三次版本迭代之后，已经可以脱离电缆，拥有液压动力，外加75公斤的质量和1.5米高的身形，拥有导航、避障能力，以及很好的平衡能力，动不动还可以来个后空翻，不得不说确实炫酷。

作为波士顿动力的明星产品，这款产品可以说是备受关注，当然，广受关注后，也必然是备受争议。不论是在后空翻的失败次数，还是在室外跟随上是否有人操作，都备受质疑。据相关媒体报道，波士顿动力一位不愿透露姓名的前员工说，视频代表的是机器人最好的状态，自从机器人开始学走路，就会经常摔倒，只是NG镜头不太会播出而已。
从技术角度来分析，不论是在平衡能力方面，还是动力配备上，在同等技术支持下，四足机器人自然会比双足机器人表现更好，如果考虑稳定性和大面积商用推广的话，四足机器人自然是更优选项。
另外，每台Atlas的造价都高达数百万美金，这也成了其商业化道路上又一大阻碍。

Spot Mini只是“更”适合商用

Spot Mini作为Spot的升级版，其是波士顿动力机器人军团中体格最小的四足机器人。
与Atlas相似，同样是采用电池能源供电，液压系统作为驱动设备，不同的是，Spot Mini只有0.85米/30公斤，充满电可以持续工作90分钟，官方给出的应用场景是：可以搬运物体，爬楼梯，并可在办公室、家中和户外操作。网传很火的Spot Mini为同伴开门，其实更多应该说是一个“精准定位+机械臂”特定场景应用。
相对于波士顿动力其他的机器人而言，Spot Mini不论是在体型上，还是在续航能力和稳定性上，相对表现更好。同时，在应用场景上也会更容易切入。

另外，Marc Raibert在CEBIT上透露：波士顿动力目前正在生产100个Spot Mini，预计在2019年底之前完成1000个产量目标。另外，Spot Mini商用后将会首先应用到娱乐、紧急救援、安全性等相关应用领域，之后也会扩展到仓储物流、包括递送，甚至施工、陪护等方面。
雷锋网小结
波士顿动力公司曾致力于军用机器人，被谷歌收购后开始转向商业化应用场景，虽然曾在谷歌大旗下研发出Spot、Atlas明星产品，但是却未能走向商用。几经转折，如今归于软银门下的波士顿公司终于对外宣布要在2019年底之前实现Spot Mini的量产和商用。
Spot Mini并不是波士顿动力机器人军团中表现最优秀的，也不是呼声最高的机器人，但是在技术成熟度、技术和产品匹配度、功耗、性价比，以及体型外观上却是最适合商业落地和场景化应用开发的机器人。
走在聚光灯下的波士顿动力终于在宣布Spot Mini商业化计划后开始走出“未来公司”这一尴尬头衔。而之所以选择Spot Mini进行商业化，除了诸多性能对比和场景化应用上的优势外，如没有新的技术赋能或产品卖点，却也难免有“矬子里面拔将军”之嫌。

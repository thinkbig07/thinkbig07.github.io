预告 | 大咖Live X 舜宇车载光学：车载传感器、智能大灯以及显示终端背后，光学技术提供了哪些助力？

科技 雷锋网 2018/08/02


8 月 8 日（星期三）晚上 8 点，【大咖Live】第 12 期视觉感知专场，我们邀请到舜宇车载光学总经理裘文伟详细解析“光学在智能驾驶中的应用和未来前景”。照例，本次分享对雷锋网新智驾会员免费开放。

背景

自动驾驶汽车能够“看见”世界，是因为激光雷达、摄像头等诸多传感器充当了“汽车之眼”。而在这些大大小小的车载传感器内部，有一些不可或缺的关键组件将对其感知效果产生决定性影响，这些组件包括镜头以及相关的光学元器件。
国内企业舜宇车载光学就是这类镜头和光学元器件的资深供应商。凭借 30 多年的光学研究经验和 14 年的车载镜头研发经验，他们深谙在自动驾驶大潮之下，如何去满足自动驾驶车辆对光学传感器的需求，特别在光学传感器如何满足车规级要求方面，有很深厚的积累。
本场 Live，舜宇车载光学总经理裘文伟将为我们阐述光学传感器在智能驾驶中的应用以及未来前景。
此外，随着智能驾驶的发展，车辆的大灯照明以及显示终端又将经历哪些变革？又有什么革新性的光学技术和产品将打入智能驾驶汽车市场？裘文伟先生也将进行深入分享。

裘文伟，长春理工大学光学工程硕士，宁波舜宇车载光学技术有限公司总经理。1997 年加入光学行业龙头企业——舜宇光学科技，成为一名优秀的光学和结构设计工程师，获得各类授权专利 17 项，其中发明专利 12 项（作为第一发明人有 9 项）。
2008 年创建车载镜头制造企业——宁波舜宇车载光学技术有限公司，产品连续 7 年雄居全球市占率之首。领衔开发的《光学镜头在自动辅助驾驶中的应用》荣获宁波市科学技术奖二等奖、浙江省技术发明奖三等奖，《光学镜头在车载主动安全领域应用的研究与产业化》荣获宁波市科学技术奖二等奖、国家科技型中小企业技术创新基金。

Q.5. Python 中如何实现多线程？
线程是轻量级的进程，多线程允许一次执行多个线程。众所周知，Python 是一种多线程语言，它有一个多线程包。
GIL（全局解释器锁）确保一次执行单个线程。一个线程保存 GIL 并在将其传递给下一个线程之前执行一些操作，这就产生了并行执行的错觉。但实际上，只是线程轮流在 CPU 上。当然，所有传递都会增加执行的开销。

Q.6. 解释继承
一个类继承自另一个类，也可以说是一个孩子类/派生类/子类，继承自父类/基类/超类，同时获取所有的类成员（属性和方法）。
继承使我们可以重用代码，并且还可以更方便地创建和维护代码。Python 支持以下类型的继承：

  1. 单继承- 一个子类类继承自单个基类
  2. 多重继承- 一个子类继承自多个基类
  3. 多级继承- 一个子类继承自一个基类，而基类继承自另一个基类
  4. 分层继承- 多个子类继承自同一个基类
  5. 混合继承- 两种或两种以上继承类型的组合
    关于继承的更多内容参考 Python Inheritance（ https://data-flair.training/blogs/python-inheritance/）

Q.7. 什么是 Flask？
Flask 是一个使用 Python 编写的轻量级 Web 应用框架，使用 BSD 授权。其 WSGI 工具箱采用 Werkzeug，模板引擎则使用 Jinja2。除了 Werkzeug 和 Jinja2 以外几乎不依赖任何外部库。因为 Flask 被称为轻量级框架。
Flask 的会话会话使用签名 cookie 来允许用户查看和修改会话内容。它会记录从一个请求到另一个请求的信息。但如果要修改会话，则必须有密钥 Flask.secret_key。
我们将在后续的课程中进一步讨论 Flask。

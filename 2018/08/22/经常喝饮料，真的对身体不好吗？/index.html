经常喝饮料，真的对身体不好吗？

今日热点 知乎日报 2018/08/22

题主说的“饮料”这个词太广了。
饮料一般可分为含酒精饮料和无酒精饮料，无酒精饮料又分成:碳酸类饮料、果蔬汁饮料、茶类饮料、咖啡饮料、乳制饮料、和功能类饮料。
我们先挑误区最大的两类说，功能类饮料和乳制饮料。
1.功能类饮料

功能饮料中的一些如某牛，氨基酸，维生素饮料等。这些运动饮料确实能短时间内使人更加兴奋清醒，特别对维持运动、恢复体力有很大作用，是一种具有广阔前景的运动营养补剂。
然而，这种饮料不能长期饮用。目前关于能量饮料的短期的好处被惊人的健康风险所抵消,如精神健康问题、冒险行为、血压升高、肥胖、牙齿侵蚀和肾脏损害等，尤其是在儿童和青少年中,健康影响最大。
虽然这些能量饮料成分稍有不同，但大多数含有水、糖、咖啡因、维生素和瓜拉纳、牛磺酸、防腐剂苯甲酸钠等。
该类饮料往往添加了咖啡因,它可以使交感神经兴奋,促进血液循环,从而达到缓解疲劳的效果。
咖啡因在适量时会有一些有益的作用,但是由于咖啡因属于兴奋剂,长期高剂量饮用会对人体造成一定的危害，可能会影响心血管系统，如血压升高。孕妇处于生理特殊时期更应谨慎选用。
对于该类饮料中添加的维生素,常常有肌醇、烟酰胺、维生素 B6 和维生素 B12。它们同属于 B 族维生素。其中,烟酰胺和肌醇属于营养强化剂。维生素 B6 和 B12 除做营养强化剂以外还辅助肌醇和烟酰胺的快速吸收,能为运动后能量流失的人尽快补充营养和能量。
维生素饮料中的维生素多是水溶性维生素，如维生素 C 或 B 族维生素，尽管水溶性维生素进入人体后可通过尿液排泄，较少发生中毒情况，但也并不意味着可以无限制摄入。
过多饮用维生素，也会造成营养过剩，对肝脏造成负担。如果你同时服用维生素补充剂，加上维生素饮料及膳食摄入的维生素，维生素的摄入就有可能过量，对身体也没什么好处。水溶性维生素之间也会有一定的相互影响，如维生素 C 过多会破坏食物中的维生素 B12，降低其生物利用率等。
而且该类饮料，有些还含有防腐剂苯甲酸钠。这类防腐剂由于含苯基,所以,用量受到严格的控制英国规定每千克软饮料中苯甲酸钠的含量<5mg,美国则规定为 1mg 我国对其在饮料中的添加用量尚没有明确的规定,由于苯甲酸钠与维生素 C，在阳光或高温条件下会生成有毒物质苯,所以消费者在购买饮料时除了应注意饮料的配料说明，也要注意是否放在背光处防止阳光直射。
总结:功能饮料一开始是为了满足某些特殊人群的需要的，比如运动员和一些高体力劳动者及时补充能量和电解质的。所以，不要为了追求所谓的功效而盲目喝功能饮料。
2.乳制品类:
普通的乳制品如牛奶酸奶确实有益青少年及成年人的身体健康，但是近几年火起来的乳酸菌饮料，却并没有它们广告说得那么有益，也无法代替牛奶、酸奶等乳制品。
这些乳酸菌营销的切入点是普及益生菌知识,很多消费者缺乏益生菌的基础知识更无从识别益生菌的优劣,就是宣传各种对胃好的养生功能。
下面说的是指活性乳酸菌饮料，不是乳酸饮料。

从成分来看，乳酸菌饮料主要是多种菌群的组合，调节肠道生态平衡，所以明显无法补充儿童成长过程中的蛋白质和钙质。
它的好处主要在于活性乳酸菌饮料可促使人体肠道内益生菌双歧杆菌和乳杆菌增殖,抑制有害菌产气荚膜梭菌的生长,具有调整人体肠道菌群的组成、改善肠道内环境的作用。
饮用含有大量益生菌的全发酵乳酸菌奶制品,不断补充乳酸菌,促进肠道蠕动，调节肠内生态平衡，该善便秘和腹泻状况。
但是，需要注意的是乳酸菌饮品并不是“0 脂肪”。虽然脂肪少，但是能量高。其次它的碳水化合物也不少。(这点高票答案已作出详尽回答。不再细述)
服用该类饮料需要注意的是冷藏，忌空腹，忌和药物同服。
再来简单说说其他几类，如茶饮类虽然功效多，但是饮用过度，其中的咖啡因也会使人过度兴奋，此外茶叶中的天然色素会导致牙齿变黄;市面上的果汁多少都含添加糖，更建议直接吃鲜果或者自己在家里榨汁。
此外，饮料中糖和水的过量的坏处也已经广为流传，因为大多饮料都含糖，能量较高，和相同数量的水相比，饮料导致人体摄入更多的热量，如果这些多余的热量不能通过运动消耗掉，就会转成脂肪储存体内，长时间下去就有可能引起超重或肥胖，并且还容易诱发龋齿。
